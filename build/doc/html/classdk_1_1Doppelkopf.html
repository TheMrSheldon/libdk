void dk::Doppelkopf::addObserver ( Observer &  observer)
noexcept

Adds a new observer to the game.

Opposite to a dk::Player an dk::Observer gets notified with all information about the currently played game. They may as such be used to implement user interfaces that output the current state of the game.

Parameters
observerThe observer that should be added.

Launches and plays an entire game of doppelkopf.

The game is played by repeatedly asking the players for their moves. When observerers were added they are notified about all changes to the game's current state.
