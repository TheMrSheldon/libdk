constexpr Card dk::Action::getPlacedCard ( ) const
inlineconstexprnoexcept

Returns the card that was placed in this action.

If this action is a placement action (if Action::getType() is equal to ActionType::PlaceCard) this function returns the card that was placed. The behavior is undefined if the action is not a placement action and this method should not be called then.

Returns
The card that is placed in this action.
