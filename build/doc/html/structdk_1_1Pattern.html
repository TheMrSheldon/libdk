Detailed Description

A pattern describes a card or cards that have a specified suit and / or value.

Generally a pattern may be understood as a card that has additional wildcard values for its suit or value. cards match a pattern iff they have the same suit or the pattern's suit is a wildcard and if they have the same value or the pattern's value is a wildcard.
